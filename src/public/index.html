Read any wallet like a health report.

Scope inspects any Ethereum address and returns a deterministic health check — concentration, stale approvals, dead tokens, and dust — then narrates it in plain English. No price predictions, no buy/sell advice, and no invented numbers.

Four checks and two stories

Every number below is computed by rule-based code — the same input always gives the same result. The two narration blocks describe those numbers; they never produce them.

◆Concentrationhealth check · numeric
How much of the wallet's liquid value sits in a single token. A one-token wallet is fragile; this surfaces it. Spam tokens with fabricated prices are excluded first, so a fake “$5M” airdrop can't distort the figure.flags at ≥ 70% in one token
◆Stale approvalshealth check · security
Token spending permissions you granted and forgot — unlimited, long-standing, and still active, or explicitly flagged vulnerable. The quiet attack surface of an old wallet.unlimited + > 180 days + active, or vulnerable
◆Dead / illiquidhealth check · market
Held tokens whose on-chain market is too thin to exit cleanly — little liquidity or almost no recent trading. Value on paper you may not be able to realize.< $10k liquidity or < $100 / 24h volume
◆Value-only dusthealth check · numeric
Sub-$2 holdings, valued only. Always shown with a mandatory disclaimer: the estimate ignores the gas cost to move it, its liquidity, and whether it's a spam or airdrop token.holdings worth < $2
▹Health summarynarration · plain English
A short paragraph over the four checks above, written for a human. The model narrates the facts it's given — it can't invent, estimate, or alter a figure.
▹Activity storynarration · plain English
A readable timeline of the wallet's notable trades and transfers over the recent window — what it did, in order, in words.

The math is deterministic. The words are just words.

Scope keeps two layers strictly separate — that separation is the whole trust model.

Analysis layer rule-based code

All numbers and flags come from deterministic, non-LLM code reading live on-chain data. Reproducible and auditable — run it twice, get the same answer.

A liquidity gate filters out spam tokens (fabricated prices, no real market) before any math, so airdropped junk can't skew the results.

Narration layer Gemini

The language model receives the already-computed facts, pre-rounded, and turns them into plain English. It is given the counts to use verbatim.

It cannot compute, estimate, or change a number — and it never gives financial advice. If narration is unavailable, the facts still return.

✓ no price predictions ✓ no buy / sell / hold language ✓ numbers never generated by the model ✓ dust always carries its disclaimer ✓ read-only — no connection, no signing
